## ✨ Fitur Utama (UI Modern)

Website portofolio ini dibangun dengan pendekatan desain UI masa depan yang berfokus pada estetika *Artificial Intelligence*.
- **Glassmorphism Design:** Elemen UI seperti kartu proyek, navigasi, dan panel profil menggunakan efek *backdrop-filter* (kaca buram).
- **Mobile-Optimized Animations:** Efek animasi *hover state* yang interaktif untuk desktop, serta performa ringan tanpa *lag* khusus untuk memuat di layar *smartphone*.
- **Dynamic Glowing Orbs:** Efek *radial gradient* cahaya di latar belakang yang memberikan kesan futuristis.
- **Dark Mode Native:** Kombinasi kontras warna neon (Ungu, Cyan) yang nyaman dipandang pada dasar warna gelap ruang angkasa yang elegan.

---

## 🛠️ Teknologi yang Digunakan

- **HTML5:** Struktur dasar yang semantik.
- **CSS3 Vanilla (Tanpa framework):** 
  - CSS Flexbox & Grid untuk *responsive layout*.
  - CSS Keyframes untuk animasi mengambang (float) dan transisi masuk.
  - CSS Variables (`:root`) untuk mempermudah palet warna dan pemeliharaan *dark theme*.
- **JavaScript (ES6):**
  - Manipulasi DOM untuk panel vertikal "About".
  - `IntersectionObserver` API untuk memuat efek *fade-in* (animasi elemen saat pertama kali di-scroll).
- **Dialogflow Messenger AI:** Integrasi *chatbot* (dapat ditemukan di menu terpisah/proyek).
- **Google Fonts:** Menggunakan *font* modern [Outfit](https://fonts.google.com/specimen/Outfit).
- **Boxicons:** *Library* ikon ringan untuk elemen visual.
